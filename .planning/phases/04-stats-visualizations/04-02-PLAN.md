**2. Add build-time stats JSON import in frontmatter (after `const { clip } = Astro.props;`):**
```javascript
// Import stats JSON at build time — Vite resolves template literal with literal prefix
const statsModule = await import(`../../../public/data/${clip.clip_id}_annotated_stats.json`);
const stats = statsModule.default;

// Extract possession percentages
const t1Pct = stats.possession.team_1_percent;
const t2Pct = stats.possession.team_2_percent;

// Partition players by team, filter ghost entries (all zeros), sort by ID ascending
const allPlayers = Object.entries(stats.players)
  .map(([id, data]) => ({ id: Number(id), ...data }))
  .filter(p => p.distance_m > 0 || p.max_speed_kmh > 0)
  .sort((a, b) => a.id - b.id);

const team1Players = allPlayers.filter(p => p.team === 1);
const team2Players = allPlayers.filter(p => p.team === 2);
```
